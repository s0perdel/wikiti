
| Code | Meaning                   | Example |
| ---- | ------------------------- | ------- |
| `%Y` | Year (4-digit)            | 2024    |
| `%y` | Year (2-digit)            | 24      |
| `%m` | Month (01-12)             | 01      |
| `%d` | Day of month (01-31)      | 15      |
| `%H` | Hour (00-23)              | 14      |
| `%M` | Minute (00-59)            | 30      |
| `%S` | Second (00-59)            | 45      |
| `%A` | Full weekday name         | Monday  |
| `%a` | Abbreviated weekday       | Mon     |
| `%B` | Full month name           | January |
| `%b` | Abbreviated month         | Jan     |
| `%p` | AM/PM                     | PM      |
| `%I` | Hour (01-12)              | 02      |
| `%j` | Day of year (001-366)     | 045     |
| `%U` | Week number (Sunday week) | 05      |
| `%W` | Week number (Monday week) | 05      |
| `%Z` | Timezone name             | PST     |
| `%z` | Timezone offset           | +0100   |
